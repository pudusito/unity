¿Qué es el Event System en Unity?

El Event System es un componente que administra:

.Eventos de entrada del usuario
.Selección y navegación de UI
.Interacciones con botones, sliders, toggles, dropdowns
.Eventos de mouse, teclado, gamepad y táctil
.Raycasts sobre UI y objetos 3D/2D interactivos

!!Se crea automáticamente cuando añades un Canvas con UI interactiva.
